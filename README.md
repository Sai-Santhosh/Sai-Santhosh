## About

I’m **Sai Santhosh V C** — an MCS student at **Rice University** (Data Science & ML) and a Data Scientist Intern in **Global Quantitative Research** at **Affinius Capital**.

I build **reliable data products** and **production ML/LLM systems** end-to-end: ingestion → modeling/retrieval → APIs → evaluation/monitoring.

## Focus

- Data engineering & analytics engineering (ETL/ELT, modeling, quality, orchestration)
- Applied machine learning (ranking/recommendation, churn/propensity, experimentation)
- LLM systems (RAG, NLQ-to-SQL, embeddings, evaluation, latency/cost tradeoffs)
- Production software (microservices, CI-friendly workflows, observability)
